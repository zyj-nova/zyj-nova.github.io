卷积操作的线性性质

卷积

我们假设有一幅大小为4 ×4 的图像 x，卷积核大小为 3×3 A

卷积核滑动步长为 1 ，没有 padding；那么经过卷积以后图像大小即为 2×2

如何将其写为矩阵相乘形式？

我们将卷积核拓展为 一个 4 × 16 的矩阵 => T，将图像x 压缩(flatten)为 16 × 1 => X

卷积操作即可表示为:

T*x = C

其中，结果 C 为4×1的矩阵。很显然，上面的这个式子是线性的；

这是矩阵和一个列向量相乘，可以理解为这个列向量经过矩阵变化之后，最终成了C

那么，这么矩阵就可以理解为列向量的一个函数。

也就是说，图像经过了卷积核这个函数，变成了最后的结果。

转置卷积

现在我们有2 × 2 的图像 x，想让他变成 4 × 4的，我们该如何操作？

我们仍然可以利用上文提到的那个卷积矩阵 T，

即：

T^T(16*4) * x(4*1) = R(16*1)

转置卷积通常用来做上采样(up sample)。
